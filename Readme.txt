
Le client JAVA SWING consomme de l'API Laravel via NET URL.
L'application manipule une base données Mysql.

Elle a les fonctionnalités suivantes:
	-Création et mise à jour des tables Client, Versement, de Retrait et éventuellement de Transfert
	-Recherche de client par son numéro de compte ou par son nom
	-Transfert d'une somme d'un compte à l'autre
